# MosHumanoid SDK

MosHumanoid SDK is a Python library for communication with MosHumanoid clients and servers.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install.

```shell
pip install moshumanoid.sdk
```
